## 기본 데이터 타입 

* Bool : 논리 자료형, true, false 값만 가짐
* Int : 정수 자료형
* UInt : 부호가 없는 정수 자료형
* Float : 실수 자료형
* Double : 실수 자료형
* Character : 문자 자료형, “” 로 표시
* String : 문자열
* Swift는 자료형 간의 직접적인 대입은 불가능 함

<pre> ex) var someFloat : Float = 3 은 가능 </pre>
<pre> ex) var someInt : Int = 3, someFloat = someInt 는 불가능 </pre>
